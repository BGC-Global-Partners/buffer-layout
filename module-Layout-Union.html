new Union(discr_layout, default_layout, propertyopt)

Represent any number of span-compatible layouts.

The span of a union is the sum of the spans of its discriminator and its default layout.

Variant layouts are added through addVariant and may be any layout that does not exceed span of the default layout.

The variant encoded in a buffer can only be identified from the content property (in the case of the default layout), or by using getVariant and examining the resulting VariantLayout instance.

registry

A registry of allowed variants.

The keys are unsigned integers which should be compatible with discr_layout. The property value is the corresponding VariantLayout instances assigned to this union by addVariant.

NOTE The registry remains mutable so that variants can be added at any time. Users should not manipulate the content of this property.
