Class: UnionLayoutDiscriminator

Layout~ UnionLayoutDiscriminator

new UnionLayoutDiscriminator(layout, offsetopt, propertyopt)

An object that can provide a discriminator API for Union using an unsigned integral Layout instance located either inside or outside the union.

Parameters:
Name Type Attributes Description
layout module:Layout~Layout

initializer for layout. Must be an instance of UInt or UIntBE.
offset Number <optional>

Initializes offset. If not provided the discriminator is assumed to be a prefix internal to the union.
property string <optional>

Default for property, superseding the property from layout, but defaulting to variant if neither property nor layout provide a property name.

Members

layout

The layout for the discriminator.

This is a UInt (or UIntBE) instance that identifies which variant is present in some Union structure for which this object serves as the discriminator.

Source:
  • Layout.js, line 761

offset

The location of the layout relative to the start of the union.

If zero the discriminator overlays the start of the union (and may be an internal discriminator). Non-zero values are used when the discriminator is external to the union or located inside the union but not at its start.
