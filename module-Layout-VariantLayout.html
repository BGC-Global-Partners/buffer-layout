Class: VariantLayout

Layout~ VariantLayout

new VariantLayout(union, variant, layout, property)

Represent a specific variant within a containing union.

NOTE The span of the variant includes the span of the discriminator used to identify it, but values read and written using the variant strictly conform to the content of layout.

NOTE User code should not invoke this construtor directly. Use the union addVariant helper method.
